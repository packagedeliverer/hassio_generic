# Test Addon

A simple test addon to verify Home Assistant addon repository functionality.

## About

This is a minimal test addon that serves a simple HTML page to verify that the addon repository is working correctly and addons can be discovered by Home Assistant.

## Features

- ✅ **Simple HTML Interface**: Basic web page to confirm functionality
- ✅ **Ingress Support**: Access through Home Assistant's ingress system
- ✅ **Multi-Architecture**: Supports ARM and x86 architectures
- ✅ **Minimal Dependencies**: Uses only nginx for serving content

## Installation

1. Add the repository: `https://github.com/packagedeliverer/hassio_generic.git`
2. Find "Test Addon" in the addon store
3. Click Install
4. Start the addon
5. Click "Open Web UI"

## Purpose

This addon exists solely to test repository functionality. If you can see this addon in the Home Assistant addon store, it means the repository is working correctly.
